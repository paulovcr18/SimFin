<objective>
Add an SRI integrity hash to the SheetJS CDN `<script>` tag in `index.html`.

Purpose: SheetJS is currently loaded from `cdn.jsdelivr.net` with no `integrity` attribute. A CDN compromise or unintended version swap could inject arbitrary code into a logged-in user's session. Adding the SRI hash ensures the browser will refuse to execute the script if its content does not match the expected hash — even if the CDN is compromised.

Output: Modified `index.html` with `integrity` and `crossorigin` attributes on the SheetJS `<script>` tag.

Note: The preload into the Service Worker cache is handled by Plan 02 (adding the URL to `LOCAL_ASSETS`). This plan only adds the SRI hash to the HTML tag.
</objective>
